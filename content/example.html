HTML Example Page

This is an example HTML page to show how the documentation system works with HTML files.

HTML Features

The system now supports both Markdown and HTML files. Here are some HTML-specific features:

  • Native HTML elements
  • Custom styling
  • JavaScript functionality

Using HTML

You can use any HTML elements in your files:

<div class="custom-class">
    <h1>Custom HTML</h1>
    <p>With any elements you want</p>
</div>

Navigation

The navigation will be automatically generated from your HTML headings, just like with Markdown files.

Best Practices

When using HTML files:

  • Use semantic HTML elements
  • Add IDs to your headings for better navigation
  • Keep the structure consistent
